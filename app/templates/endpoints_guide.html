Guia Amigavel de Endpoints

Documentacao rapida para uso manual e integracao com ESP32/app. Esta pagina foca nos endpoints mais usados do projeto.

OpenAPI /docs OpenAPI /redoc Preview Painel Preview Frame
GET /health

Health check simples para monitoramento.

curl http://127.0.0.1:8000/health
GET /

Resumo do servico e atalhos principais.

curl http://127.0.0.1:8000/
GET /screen?img_mode=rgb565_base64

Payload ativo escolhido pelo gerenciador de widgets (spotify, book, clock).

img_mode: rgb565_base64 (padrao), rgb_base64, rgb_array

curl "http://127.0.0.1:8000/screen?img_mode=rgb565_base64"
GET /screen/frame?at_ms=<ms>&refresh_source=false

Frame 64x32 pronto para renderizacao (RGB565 base64), incluindo metadados de cache/fonte.

curl "http://127.0.0.1:8000/screen/frame?refresh_source=true"
GET /book/current

Estado atual do widget de livro.

curl http://127.0.0.1:8000/book/current
POST /book/current

Atualiza estado do livro para forcar exibicao quando desejado.

curl -X POST "http://127.0.0.1:8000/book/current" \
  -H "Content-Type: application/json" \
  -d '{
    "is_reading": true,
    "title": "Clean Code",
    "author": "Robert C. Martin",
    "cover_url": "https://exemplo/capa.jpg"
  }'
GET /preview/painel e /preview/frame

Paginas de preview visual para validacao sem hardware.

http://127.0.0.1:8000/preview/painel
http://127.0.0.1:8000/preview/frame
GET /config/widgets e /widgets/config

Tela amigavel para escolher widgets ativos e endpoint JSON para leitura da configuracao.

http://127.0.0.1:8000/config/widgets
curl http://127.0.0.1:8000/widgets/config
POST /widgets/config

Atualiza os widgets habilitados no backend.

curl -X POST "http://127.0.0.1:8000/widgets/config" \
  -H "Content-Type: application/json" \
  -d '{"enabled_widgets": ["spotify", "clock"]}'

Se precisar da documentacao completa com schemas, use /docs ou /redoc.
