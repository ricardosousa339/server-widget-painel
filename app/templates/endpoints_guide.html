Guia Amigavel de Endpoints

Documentacao rapida para uso manual e integracao com ESP32/app. Esta pagina foca nos endpoints mais usados do projeto.

OpenAPI /docs OpenAPI /redoc Preview Painel Preview Frame
GET /health

Health check simples para monitoramento.

curl http://127.0.0.1:8000/health
GET /

Resumo do servico e atalhos principais.

curl http://127.0.0.1:8000/
GET /screen?img_mode=rgb565_base64

Payload ativo escolhido pelo gerenciador de widgets (spotify, custom_gif, clock) e sobrescrito pela janela temporaria da campainha quando ela estiver ativa.

img_mode: rgb565_base64 (padrao), rgb_base64, rgb_array

curl "http://127.0.0.1:8000/screen?img_mode=rgb565_base64"
GET /screen/frame?at_ms=<ms>&refresh_source=false

Frame 64x32 pronto para renderizacao (RGB565 base64), incluindo metadados de cache/fonte.

curl "http://127.0.0.1:8000/screen/frame?refresh_source=true"
GET /preview/painel e /preview/frame

Paginas de preview visual para validacao sem hardware.

http://127.0.0.1:8000/preview/painel
http://127.0.0.1:8000/preview/frame
GET /config/widgets e /widgets/config

Tela amigavel para escolher widgets ativos, gerenciar a biblioteca de GIFs custom e configurar o GIF separado da campainha.

http://127.0.0.1:8000/config/widgets
curl http://127.0.0.1:8000/widgets/config
POST /widgets/config

Atualiza os widgets habilitados no backend.

curl -X POST "http://127.0.0.1:8000/widgets/config" \
  -H "Content-Type: application/json" \
  -d '{"enabled_widgets": ["spotify", "custom_gif", "clock"]}'
GET /widgets/custom-gif, /widgets/custom-gif/raw e /widgets/custom-gif/playback

Consulta a biblioteca atual, baixa o GIF bruto do asset selecionado e recupera o pacote de playback pronto para o ESP32.

curl http://127.0.0.1:8000/widgets/custom-gif
curl -OJ "http://127.0.0.1:8000/widgets/custom-gif/raw?kind=custom"
curl -OJ "http://127.0.0.1:8000/widgets/custom-gif/playback?kind=custom"
POST /widgets/custom-gif/upload

Faz upload de um GIF para a biblioteca custom ou para o GIF da campainha. Aceita `kind` e `active` no form multipart.

curl -X POST "http://127.0.0.1:8000/widgets/custom-gif/upload" \
  -F "file=@/caminho/animacao.gif" \
  -F "kind=custom" \
  -F "active=true"
PATCH /widgets/custom-gif/{asset_id}

Ativa ou desativa um asset existente sem reenviar o arquivo.

curl -X PATCH "http://127.0.0.1:8000/widgets/custom-gif/ASSET_ID" \
  -H "Content-Type: application/json" \
  -d '{"active":false}'
DEL /widgets/custom-gif

Remove um asset especifico ou limpa a biblioteca inteira do kind indicado.

curl -X DELETE "http://127.0.0.1:8000/widgets/custom-gif/ASSET_ID"
curl -X DELETE "http://127.0.0.1:8000/widgets/custom-gif?kind=custom"
GET /integrations/doorbell/state

Estado atual da janela de alerta temporaria da campainha (TTL).

curl http://127.0.0.1:8000/integrations/doorbell/state
POST /integrations/doorbell/trigger

Dispara alerta de campainha e prioriza o GIF separado da campainha por alguns segundos.

curl -X POST "http://127.0.0.1:8000/integrations/doorbell/trigger" \
  -H "Content-Type: application/json" \
  -d '{"duration_seconds":8,"source":"home_assistant"}'
DEL /integrations/doorbell/state

Encerra imediatamente a janela de alerta de campainha.

curl -X DELETE "http://127.0.0.1:8000/integrations/doorbell/state"
